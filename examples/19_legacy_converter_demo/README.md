# Legacy Converter Demo

**Level 5: Showroom**

PHP/JS to Omni ingestion demonstration.

## Supported Modes

| Mode        | Supported | What Happens            |
| ----------- | --------- | ----------------------- |
| `--cmd`     | ✅ Yes    | Shows conversion output |
| `--app`     | ❌ No     | Terminal only           |
| `--web`     | ❌ No     | Terminal only           |
| `--web-app` | ❌ No     | Terminal only           |

## Run Command

```powershell
.\omni run examples/19_legacy_converter_demo/src/main.omni --cmd
```

## What It Demonstrates

- Legacy code ingestion concepts
- PHP to Omni conversion
- Code transformation pipeline
