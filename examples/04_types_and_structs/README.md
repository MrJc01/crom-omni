# Types and Structs

**Level 1: Core Fundamentals**

Demonstrates the Omni type system.

## Supported Modes

| Mode        | Supported | What Happens     |
| ----------- | --------- | ---------------- |
| `--cmd`     | ✅ Yes    | Runs in terminal |
| `--app`     | ❌ No     | Terminal only    |
| `--web`     | ❌ No     | Terminal only    |
| `--web-app` | ❌ No     | Terminal only    |

## Run Command

```powershell
.\omni run examples/04_types_and_structs/src/main.omni --cmd
```

## What It Demonstrates

- Primitive types: `i64`, `f64`, `bool`, `string`
- `struct` definitions
- Nested structs
- Struct initialization syntax
