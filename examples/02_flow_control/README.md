# Flow Control

**Level 1: Core Fundamentals**

Demonstrates control flow structures in Omni.

## Supported Modes

| Mode        | Supported | What Happens                   |
| ----------- | --------- | ------------------------------ |
| `--cmd`     | ✅ Yes    | Runs in terminal, shows output |
| `--app`     | ❌ No     | Terminal only (no GUI)         |
| `--web`     | ❌ No     | Terminal only                  |
| `--web-app` | ❌ No     | Terminal only                  |

> **Note:** This is a terminal-only example. It demonstrates language features, not visual output.

## Run Command

```powershell
.\omni run examples/02_flow_control/src/main.omni --cmd
```

## What It Demonstrates

- `if` / `else if` / `else` conditionals
- `while` loops
- Recursion (factorial)
- Comparison operators
- FizzBuzz algorithm
