# JSON Transformer

**Level 2: Integration**

Demonstrates JSON parsing and transformation.

## Supported Modes

| Mode        | Supported | What Happens               |
| ----------- | --------- | -------------------------- |
| `--cmd`     | ✅ Yes    | Parses JSON, prints output |
| `--app`     | ❌ No     | Terminal only              |
| `--web`     | ❌ No     | Terminal only              |
| `--web-app` | ❌ No     | Terminal only              |

## Run Command

```powershell
.\omni run examples/07_json_transformer/src/main.omni --cmd
```

## What It Demonstrates

- `json_parse()` function
- `json_stringify()` function
- Object manipulation
