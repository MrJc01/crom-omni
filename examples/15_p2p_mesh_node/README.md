# P2P Mesh Node

**Level 4: Systems & Networks**

P2P mesh networking demonstration.

## Supported Modes

| Mode        | Supported | What Happens                    |
| ----------- | --------- | ------------------------------- |
| `--cmd`     | ✅ Yes    | Initializes node, prints status |
| `--app`     | ❌ No     | Terminal only                   |
| `--web`     | ❌ No     | Terminal only                   |
| `--web-app` | ❌ No     | Terminal only                   |

## Run Command

```powershell
.\omni run examples/15_p2p_mesh_node/src/main.omni --cmd
```

## What It Demonstrates

- Peer ID generation
- Network node initialization
- P2P architecture concepts
